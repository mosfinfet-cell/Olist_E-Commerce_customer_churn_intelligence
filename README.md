# Olist_E-Commerce_customer_churn_intelligence
A 5-stage product analytics deep dive using the Olist E-Commerce Dataset. This project moves from exploratory data analysis to RFM segmentation, cohort retention modeling, churn diagnosis, and finally a PM-style product roadmap.

I analysed 100k+ orders from a Brazilian e-commerce platform using RFM, cohort analysis, and hypothesis testing. I found that later cohorts retain at half the rate of earlier ones — structural deterioration, not noise. Four evidence-ranked hypotheses drive eight product interventions. Estimated revenue opportunity: 30% conversion. Full analysis in this PRD.
